# Meu fluxo de trabalho como Engenheiro de Qualidade na Era da IA

Apresentação da oficina sobre engenharia de software + IA, construída com [Slidev](https://sli.dev).

## Executar

```bash
cd app/slidev
pnpm install
pnpm dev
```

Abre em `http://localhost:3030`.

## Exportar PDF

```bash
cd app/slidev
pnpm add -D playwright-chromium
pnpm export
```

## Tema

Tema [Dracula](https://github.com/jd-solanki/slidev-theme-dracula) (`theme: dracula` no `slides.md`).

## Personalizar

Edite `app/slidev/slides.md` para ajustar conteúdo e slides. Foto de perfil em `app/slidev/public/eu.jpeg`.
